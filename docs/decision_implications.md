# Decision Implications

## Purpose

This note translates the descriptive visitor-retail findings into monitoring implications. It does not provide causal policy evaluation or operational recommendations.

## Practical Reading

1. Total retail recovery should not be treated as a sufficient signal for visitor-facing retail recovery.
2. Tourist-sensitive categories should be monitored separately from total retail.
3. Local daily and durable/household groups should be interpreted cautiously because they are more baseline-sensitive.
4. Category grouping assumptions should be checked because the tourist-sensitive definition involves judgment.
5. The grouping robustness layer reduces, but does not remove, category-definition risk.

The grouping robustness check is directionally consistent with the main descriptive finding: current, strict, and broad tourist-sensitive definitions all show negative recent-adjustment average gaps. This should be read as a robustness signal, not as causal evidence.

## What This Analysis Can Support

It can support:

- Descriptive monitoring.
- Category-level recovery tracking.
- Dashboard-based reporting.
- Identifying where further investigation is needed.

## What This Analysis Cannot Support

It cannot support:

- Causal claims.
- Claims about individual tourist spending behavior.
- Firm-level recommendations.
- Transaction-level conclusions.

## Current Best Use

Use this project as a monitoring framework for comparing visitor recovery with retail category recovery, not as a final decision model.
